# EmberLearn Root Generator

## When to Use
- Initializing a fresh project structure
- Regenerating core configuration files
- Setting up the development environment from scratch

## Instructions
1. Run the generation script:
   - `python scripts/generate_root.py`

## Output
- `package.json`
- `.gitignore`
- `README.md`
- `setup.sh`
- `start.sh`
- `reseed_db.py`
- `.env.example`
